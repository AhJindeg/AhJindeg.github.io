## 片段

前缀 `gitignore` 展开为常用忽略项（含 `!pnpm-lock.yaml` 等）。

```json
{
  "Gitignore": {
    "prefix": "gitignore",
    "body": [
      "*.DS_Store",
      ".idea/",
      "",
      "node_modules",
      "dist-ssr",
      "dist",
      "",
      "*.local",
      "*-lock.*",
      "!pnpm-lock.yaml",
      "*.lock",
      "*.log",
      "",
      "auto-*.d.ts",
    ],
    "description": "gitignore file"
  }
}
```
