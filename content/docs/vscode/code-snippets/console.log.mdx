## 片段

将以下 JSON 合并进 VS Code **用户代码片段**（`javascript.json` 或 `*.code-snippets`）后，可用前缀 `consolelog` / `consoletable` / `consoledir` 快速插入。

```json
{
  "Console Log": {
    "body": [
      "console.log(`%c$TM_FILENAME:%c$TM_LINE_NUMBER: `, `color: ${9:green};`,`color: ${10:red};`, {",
      " ${1} ",
      "})"
    ],
    "description": "Console Log",
    "prefix": "consolelog",
  },
  "Console Table": {
    "body": [
      "console.table(${1}, [${2}])"
    ],
    "description": "Console Table",
    "prefix": "consoletable",
  },
  "Console Dir": {
    "body": [
      "console.dir(${1}, { depth: ${2:2} })"
    ],
    "description": "Console Dir",
    "prefix": "consoledir",
  },
}
```
