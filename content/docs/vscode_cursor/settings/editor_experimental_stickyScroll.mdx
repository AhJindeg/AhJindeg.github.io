title: 顶部浮动显示当前层级
description: 开启 editor.experimental.stickyScroll，长文件滚动时固定显示当前类/函数
---

## 说明

较新版本 VS Code 支持 **粘性滚动**（Sticky Scroll）：在编辑器中向下滚动时，在**顶部**固定显示当前所在的**类/函数/命名空间**层级，便于长文件内定位结构。

```json
"editor.experimental.stickyScroll.enabled": true
```
