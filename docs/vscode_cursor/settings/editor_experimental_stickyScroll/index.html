顶部浮动显示当前层级

开启 editor.experimental.stickyScroll，长文件滚动时固定显示当前类/函数

说明

较新版本 VS Code 支持 粘性滚动（Sticky Scroll）：在编辑器中向下滚动时，在顶部固定显示当前所在的类/函数/命名空间层级，便于长文件内定位结构。
